## What is ASWA?

ASWA is a **mobile-first digital ordering platform** built for La Rica Chicha, a traditional artisan corn chicha business in Tarapoto and Morales, Peru. It brings modern e-commerce tools — interactive catalog, real-time order tracking, delivery management, and customer loyalty — to a small local business through a zero-install progressive web app.

Customers open the page on their phone, select their chicha, enter their address, choose a payment method, and the order flows directly to the business via WhatsApp — no app download required.

## Architecture overview

ASWA is a **single-page application** built with vanilla HTML, CSS, and JavaScript. It uses Firebase as its backend for authentication, database, and file storage.

```
index.html          ← Main ordering app (SPA with 4 tab views)
ugc.html            ← UGC submission & gallery page
js/
  app.js            ← Cart, LoyaltySystem, Missions classes
  config.js         ← Firebase environment configuration
  firebase.js       ← Firebase initialization
  osm-maps.js       ← OpenStreetMap integration
css/
  styles.css        ← Global stylesheet
```

The Firebase project (`pedidos-aswa-peru`) powers:

- **Firestore** — Orders, customers, business config, UGC
- **Authentication** — Anonymous sign-in for frictionless ordering
- **Storage** — UGC photo uploads

## Multi-business vision

La Rica Chicha is the **first deployed instance** of ASWA. The platform is architected for multi-business expansion: each business is identified by a `BUSINESS_ID` constant, and all business-specific data (name, WhatsApp numbers, pricing) is stored in the `negocios/{businessId}` Firestore document.

<Info>
  The roadmap includes serving each business from a subdomain (e.g., `negocio.laricachicha.com`) or URL parameter (`?b=negocio001`), with a central admin dashboard for onboarding new businesses.
</Info>
